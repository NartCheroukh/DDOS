Python HTTP Stress Test Tool

A simple Python-based HTTP request stress testing tool created for educational and learning purposes.
This project demonstrates basic concepts of multithreading, HTTP requests, and network testing using Python.

Features

Simple command-line interface

Multi-threaded request sending

Custom target URL input

Adjustable request/thread amount

Basic error handling

Requirements

Python 3.x

requests

colorama

Install dependencies:

pip install requests colorama
Usage

Run the tool:

python ddos.py

Then enter:

Target URL

Number of request threads
